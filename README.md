# White Mirror Protocol v2.1

A deployment-ready interactive dashboard for the White Mirror Protocol, presenting the recursive hermeneutic framework as a lightweight web experience.

## Run locally

```bash
cd /workspaces/White-Mirror
npm start
```

Then open http://localhost:3000.

## What is included

- A polished dashboard interface for the protocol concept
- Live metric cards and self-application analysis controls
- A simple API for metrics and recursive analysis calculations

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import it into Vercel.
3. Deploy the project.
4. Open the generated URL.

The dashboard will be available at the Vercel URL, and the local route remains at http://localhost:3000/white-mirror-dashboard.
